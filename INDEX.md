## 📝 Quick Reference

### Common Tasks

**I want to change colors**
→ Edit `src/App.css` lines 7-20

**I want to change text**
→ Edit files in `src/components/`

**I want to speed up animations**
→ Change `duration` in component transitions

**I want to add a new section**
→ Copy a component as template, add to `src/App.jsx`

**I want to deploy**
→ Run `npm run build`, see README.md

**I want to change animation speed**
→ Find `transition={{ duration: 0.X }}` and adjust
